Configuración de la Aplicación (PWA Gastos Hogar)

Este documento detalla la configuración necesaria para las integraciones y variables del sistema.

1. Google Console Configuration (google_config.json)

Para habilitar la autenticación de Google, se deben configurar las siguientes variables:

{
  "google_client_id": "TU_CLIENT_ID.apps.googleusercontent.com",
  "google_client_secret": "TU_CLIENT_SECRET",
  "authorized_redirect_uris": [
    "https://tu-app.web.app/auth/google/callback",
    "http://localhost:3000/auth/google/callback"
  ],
  "scopes": [
    "openid",
    "profile",
    "email"
  ]
}

2. Configuración de Moneda (currency_config.json)

Lista de monedas soportadas por la aplicación. Se pueden añadir más siguiendo este formato:

{
  "default_currency": "CLP",
  "available_currencies": [
    { "code": "CLP", "symbol": "$", "name": "Peso Chileno" },
    { "code": "ARS", "symbol": "$", "name": "Peso Argentino" },
    { "code": "USD", "symbol": "$", "name": "Dólar Estadounidense" },
    { "code": "EUR", "symbol": "€", "name": "Euro" }
  ]
}

3. Configuración de Categorías

Categorías base para gastos y compromisos:
Vivienda
Alimentación
Transporte
Ocio
Salud
Servicios (Luz, Agua, Internet)
Otros (Permite añadir nuevas mediante el botón "Añadir")

4. Importación de Datos

Formatos soportados para la carga masiva:
JSON: Estructura de array de objetos con fecha, monto, nombre y categoria.
XML: Formato estándar de exportación bancaria.
Excel/CSV: Columnas predefinidas para mapeo automático.
